# Голосовой ввод с Groq Whisper

## Что было добавлено

1. **Кнопка голосового ввода** с иконкой арфы слева от кнопки отправки
2. **Анимация пульсации** при записи (имитация звуковых волн)
3. **Интеграция с Groq Whisper API** для распознавания речи

## Как это работает

1. Пользователь нажимает на кнопку с арфой
2. Начинается запись аудио (появляется пульсация)
3. При повторном нажатии запись останавливается
4. Аудио отправляется на сервер для транскрипции
5. Распознанный текст появляется в поле ввода

## Настройка на сервере

### 1. Установите зависимости

```bash
npm install groq-sdk formidable
```

### 2. Создайте файл `api/transcribe.js`

Скопируйте содержимое из `api-transcribe-example.js` в ваш Vercel проект.

### 3. Добавьте переменную окружения

В настройках Vercel проекта добавьте:
- `GROQ_API_KEY` - ваш API ключ от Groq

### 4. Деплой

```bash
vercel --prod
```

## Анимация пульсации

Используется CSS анимация `voicePulse`:
- Две волны с задержкой 0.75s
- Скругленная прямоугольная обводка
- Плавное расширение и fade-out

## Поддержка браузеров

Требуется поддержка:
- `navigator.mediaDevices.getUserMedia()` - для записи аудио
- `MediaRecorder API` - для захвата аудио
- `FormData` - для отправки файла

Поддерживается в современных браузерах (Chrome, Firefox, Safari, Edge).
